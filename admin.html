📋 Excel Format Required:

Columns (in this exact order):

Notes:

  • correct_answer should be 1-5 (1=A, 2=B, 3=C, 4=D, 5=E)
  • test_id should be 1-10 depending on which test
  • Each test typically contains 40-42 questions
  • Make sure all tests (1-10) exist in database before importing
